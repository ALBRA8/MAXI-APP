# Política de Privacidad

**Última actualización: 11 de marzo de 2026**

### 1. Información que recopilamos
CODEX recopila únicamente la información necesaria para el funcionamiento de los servicios autorizados a través de las APIs integradas. Esto incluye datos de configuración del usuario, consultas enviadas al sistema para el procesamiento de tareas y registros de actividad para la mejora de la estabilidad.

### 2. Uso de la información
Los datos recopilados se utilizan exclusivamente para:
* Ejecutar las funciones automatizadas solicitadas por el usuario.
* Mantener la seguridad y operatividad del sistema.
* Garantizar el cumplimiento de las políticas de las plataformas integradas (Meta/Facebook).

### 3. Protección de datos
CODEX se compromete a no compartir, vender ni distribuir información personal de los usuarios con terceros, salvo requerimiento legal. Toda interacción se realiza mediante conexiones seguras.

### 4. Derechos del usuario
El usuario tiene derecho a solicitar la eliminación de sus datos de configuración y registros de actividad en cualquier momento.

### 5. Contacto
Para cualquier duda sobre esta política, comuníquese a través del canal de soporte oficial del repositorio.
